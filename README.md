<h2> WS-TP2-NetBeans-GlassFish :</h2>
<h3>Développement d’un Service Web avec NetBeans et GlassFish</h3>
Ce travail vise à créer un <b>service web</b>
-ayant pour but d’effectuer une simple opération d’ajout- 
avec <b>IDE NetBeans</b> et le Serveur <b>GlassFish</b> et le tester, 
puis le consommer à partir de deux clients : <b>Application JAVA</b> et <b>Application Web</b>.
